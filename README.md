# Google Play Store App Analysis – Power BI

## Project Overview
This project analyzes Google Play Store application data to identify key factors influencing app success, user satisfaction, and download trends.

## Objectives
- Analyze app ratings and installs
- Identify top-performing categories
- Study the impact of app size and price
- Understand user preferences and content ratings

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Pages

### Executive Dashboard
- Average Rating
- Total Installs
- Total Reviews
- Total Apps
- Category Analysis
- Free vs Paid Apps

### Business Analysis
- Installs vs Rating
- Price vs Rating
- Size vs Installs
- Top Apps by Reviews
- Update Trends

### Insights & Recommendations
- Key business findings
- Strategic recommendations

## Key Findings
1. Free apps dominate the Play Store.
2. Highly rated apps generally receive more installs.
3. Certain categories drive most downloads.
4. App size has limited impact on popularity.
5. Frequent updates improve competitiveness.
